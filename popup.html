About This Extension

This extension shows bundle information from TrenchBot when viewing tokens on Axiom Trading Platform.

Key features:

  • View bundle percentage and holding data
  • Interactive bubble visualization of wallet distribution
  • Filter by wallet type (sniper, regular, etc.)
  • Explore detailed wallet information with Solscan links

Note: Navigate to any token page on Axiom to view bundle statistics automatically.
